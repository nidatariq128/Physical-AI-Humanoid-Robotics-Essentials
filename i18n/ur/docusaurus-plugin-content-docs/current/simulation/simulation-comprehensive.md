# روبوٹ محاکات

## مجازی روبوٹس

ایک مجازی روبوٹ ایک حقیقی روبوٹ کا کمپیوٹر ماڈل ہے۔ آپ بغیر حقیقی روبوٹس کو توڑے محفوظ طریقے سے ٹیسٹ کر سکتے ہیں۔

## گیزبو محاکات

گیزبو روبوٹس کے لیے ایک 3D محاکاتی پروگرام ہے۔

### گیزبو انسٹال کریں
```bash
sudo apt install ros-humble-gazebo-*
```

## روبوٹ ماڈلز

روبوٹس URDF فائلز میں تفصیل دی گئی ہے:

```xml
<robot name="simple_robot">
  <link name="base_link">
    <visual>
      <box size="0.5 0.5 0.2"/>
    </visual>
  </link>
</robot>
```

## محاکات میں فزکس

گیزبو ماحاکات کرتا ہے:
- گریویٹی
- کولیژنز
- فریکشن

## محاکات میں سینسرز

مجازی روبوٹس کے پاس مجازی سینسرز ہو سکتے ہیں:
- کیمرے
- لیزر
- IMU

## یونٹی محاکات

یونٹی دیگر محاکاتی اختیار ہے۔

## وژولائزیشن

- انٹرایکشن کے لیے گیزبو گرافیکل یوزر انٹرفیس
- ROS وژولائزیشن کے لیے RViz2

## بہترین طریقے

- سادہ شروع کریں
- معمول کو تدریج سے شامل کریں

## خلاصہ

روبوٹس کو ٹیسٹ کرنے کے لیے محاکات محفوظ ہے۔ گیزبو ایک مقبول اختیار ہے۔
