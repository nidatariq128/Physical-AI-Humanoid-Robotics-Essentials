# کیپ اسٹون: مکمل سسٹم انضمام

## جائزہ

کیپ اسٹون مodule تمام پچھلے سبسٹم کو ایک مکمل مصنوعی ذہانت روبوٹکس سسٹم میں ضم کرتا ہے۔ یہ جامع سسٹم ڈیجیٹل ذہانت کو جسمانی روبوٹک صلاحیتوں کے ساتھ جوڑ کر جسمانی مصنوعی ذہانت کی مکمل صلاحیت کو ظاہر کرتا ہے۔

## سیکھنے کے اہداف

اس کیپ اسٹون مodule میں، آپ سیکھیں گے:
- تمام اہم سبسٹم کو ایک مربوط سسٹم میں ضم کرنا
- کارکردگی کی کمی کے نظم کے ساتھ سیم ٹو ریل ٹرانسفر کو انجام دینا
- مکمل سسٹم کی کارکردگی کی توثیق کرنا
- جامع کیپ اسٹون پروجیکٹ مکمل کرنا

## ضروریات

اس مodule کا آغاز کرنے سے پہلے، یقین کر لیں کہ آپ نے مکمل کر لیا ہے:
- بنیادی مodule
- ROS 2 کنٹرول سسٹم مodule
- سیمیولیشن مodule
- ادراک اور نیویگیشن مodule
- وژن-زبان-ایکشن سسٹم مodule

## مodule کی ساخت

یہ مodule ان پر مشتمل ہے:
1. سسٹم انضمام ورک فلو
2. سیم ٹو ریل ٹرانسفر کی تکنیکیں
3. کارکردگی کی کمی کا نظم
4. مکمل کیپ اسٹون پروجیکٹ کا نفاذ

## خلاصہ

کیپ اسٹون مodule اس کتاب میں آپ کے تمام سیکھنے کی اہمیت رکھتا ہے۔ اس مodule کو مکمل کرنے سے، آپ ایک مکمل مصنوعی ذہانت روبوٹکس سسٹم تیار کر چکے ہوں گے جو قدرتی زبان کے کمانڈز وصول کرنے، اس کے ماحول کو سمجھنے، محفوظ طریقے سے نیویگیٹ کرنے، اور پیچیدہ مینوپولیشن ٹاسکس انجام دینے کا قابل ہوگا۔
